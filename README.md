> 从源码开始：

   ```
   sudo apt install -y libbz2-dev libffi-dev libssl-dev
   ```

   ```
   ./configure --enable-optimizations
   ```

   ```
   make
   ```

   ```
   make install
   ```

> 查看版本:

   ```
   python3 --version
   ```

> 执行代码:

   ```
   python3 main.py
   ```

> 仅检查语法，不执行代码:

   ```
   pyflakes main.py
   ```
   
> 格式化代码：

   ```
   black main.py
   ```
